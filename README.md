# 📊 EDA Case Study: Global Commodity Prices (1960–2022)

A comprehensive **Exploratory Data Analysis (EDA)** project analyzing **62 years of monthly global commodity prices** to uncover long-term trends, price volatility, seasonality, and relationships among energy, agricultural, and beverage commodities using Python.

---

# 📌 Project Overview

Commodity prices influence economies worldwide by affecting inflation, international trade, agriculture, and energy markets.

In this project, I performed Exploratory Data Analysis (EDA) on historical commodity prices from **January 1960 to December 2022** to identify trends, compare commodity movements, and derive meaningful insights through statistical analysis and visualization.

---

# 🎯 Objectives

* Understand the structure of the dataset
* Analyze long-term commodity price trends
* Compare price movements across commodities
* Detect correlations between commodities
* Explore price distributions and outliers
* Generate business and economic insights through visualizations

---

# 📂 Dataset

**Dataset:** Global Commodity Prices (1960–2022)

### Dataset Summary

* 📅 Monthly observations from **January 1960 to December 2022**
* 🌍 Global commodity prices
* 📈 **756 monthly records**
* 📦 **10 major commodities**
* ⏳ Covers **62 years** of historical data

### Dataset Columns

| Column            | Description               |
| ----------------- | ------------------------- |
| `date`            | Observation date          |
| `oil_brent`       | Brent crude oil price     |
| `oil_dubai`       | Dubai crude oil price     |
| `coffee_arabica`  | Arabica coffee price      |
| `coffee_robustas` | Robusta coffee price      |
| `tea_columbo`     | Colombo tea auction price |
| `tea_kolkata`     | Kolkata tea auction price |
| `tea_mombasa`     | Mombasa tea auction price |
| `sugar_eu`        | European sugar price      |
| `sugar_us`        | United States sugar price |
| `sugar_world`     | World sugar price         |

# 🔍 Key Insights

* Brent and Dubai crude oil prices exhibit very similar long-term trends.
* Arabica and Robusta coffee prices show noticeable periods of volatility.
* Tea prices vary across the Colombo, Kolkata, and Mombasa auction markets.
* Sugar prices differ between the European, U.S., and global markets.
* Correlation analysis highlights strong relationships among commodities within the same category.
* Historical trends reveal periods of significant price fluctuations influenced by global market conditions.
